Coding Portfolio

This is a space for me to host and display a couple of the projects I've been working on.
As I get more adept at coding, hopefully these will become increasingly impressive.

The below projects were constructed as part of The Complete Web Development Bootcamp course on Udemy. Credit goes to Angela Yu for designing the course, many of the resources, and teaching me how to do all of this!

The Dice Game

Built using Javascript, a game to help you settle arguments. The game rolls a die for each player, with the highest number winning.
Makes use of a random number generator and using variables to select items.

Electronic Drumkit

Built using Javascript, an Electronic Drumit. The kit responds to both button and key presses, generating a different sound for each drum.
Makes use of a event listeners, as well as mapping audio responses to events.

The Simon Game

Built using Javascript and jQuery, a digital copy of the Simon Electronic Game. The game starts by flashing a coloured button and making a sound. Press the coresponding button to progress to the next level. Each level adds a colour to the pattern, creating a longer pattern for you to memorise and repeat.
Makes use of array variables, higher order functions, and loops. Not very mobile friendly.
